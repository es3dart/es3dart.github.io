This is the supplementary material for the paper

Each folder corresponds to an example in the paper, as suggested in the folder name. Each folder follows the file structure below:
  - films: rendered film at each camera
  - viz*: visualization of the 3D object, where
    - viz.mp4      : the 360-degree video.
    - viz/*.png    : the same video's all individual frames.
    - viz_grid.png : the same video's frames, organized as a grid.
